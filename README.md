# Как запустить бота-трекера (без программирования)

## Шаг 1. Создать бота в Telegram

1. Открой Telegram, найди **@BotFather**.
2. Напиши ему `/newbot`.
3. Придумай имя (любое, например "Мой трекер") и username, заканчивающийся на `bot` (например `arthur_tracker_bot`).
4. BotFather пришлёт тебе **токен** — длинная строка вида `7234567890:AAH...`. Сохрани её, она понадобится дальше.

## Шаг 2. Узнать chat_id всех, кому нужен доступ

1. Каждый человек, которому нужен доступ к боту, находит в Telegram **@userinfobot** и пишет ему любое сообщение.
2. Бот пришлёт число — это `chat_id` этого человека. Собери все числа (через запятую), например: `111111111,222222222`.

Теперь у всех, чьи chat_id указаны, будет общий доступ: каждый может добавлять события (`/new`), видеть общий список (`/list`), и каждый получит все напоминания. Нажатие кнопки (Сделано/Отменить/Перенести) одним человеком обновит статус события для всех, но сообщение с кнопками у остальных просто останется как было показано — это нормально, событие в базе всё равно обновится.

## Шаг 3. Разместить бота на Railway

1. Зайди на **https://railway.app** и зарегистрируйся (проще всего через GitHub-аккаунт — если его нет, заведи на github.com, это бесплатно и займёт 2 минуты).
2. На GitHub создай новый репозиторий (кнопка "New repository"), назови, например, `calendar-bot`.
3. Загрузи туда 3 файла, которые я подготовил: `bot.py`, `requirements.txt`, `Procfile` (через кнопку "Add file" → "Upload files" на странице репозитория — просто перетаскиваешь файлы мышкой).
4. Вернись в Railway, нажми **"New Project"** → **"Deploy from GitHub repo"** → выбери свой репозиторий `calendar-bot`.
5. Railway сам поймёт, что это Python-проект, и начнёт сборку.

## Шаг 4. Указать токен и chat_id

1. В Railway зайди в свой проект → вкладка **"Variables"**.
2. Добавь две переменные:
   - `TELEGRAM_BOT_TOKEN` = токен из Шага 1
   - `ALLOWED_CHAT_IDS` = список chat_id через запятую, без пробелов, например `111111111,222222222`
3. Сохрани — Railway автоматически перезапустит бота с этими настройками.

## Шаг 5. Проверка

Напиши своему боту в Telegram:
```
/start
```
Он должен ответить инструкцией. Дальше добавляй события так:
```
/new Созвон с Гузель | 03.07.2026 | 15:00
```
Бот сам напомнит за 12 часов, 2 часа, 1 час и 15 минут до события, и под каждым напоминанием будут кнопки: ✅ Сделано, ❌ Отменить, ⏰ +30 минут, 📅 +1 день.

Посмотреть список активных событий: `/list`
